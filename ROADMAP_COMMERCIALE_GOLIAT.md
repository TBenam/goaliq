# Feuille de route commerciale Goliat

## Vision

Goliat doit devenir la reference francophone africaine pour les parieurs qui veulent un systeme complet:

- pronostics IA
- suivi de bankroll
- tickets VIP
- historique verifie
- accompagnement WhatsApp
- CRM de relance et retention

## Promesse de vente

Ne pas vendre "des gains garantis".

Vendre plutot:

> Un systeme VIP pour prendre de meilleures decisions, suivre ses tickets et progresser avec discipline.

## Roadmap 30 jours

1. Brancher les relances WhatsApp semi-automatiques.
2. Ajouter un bouton "Copier message de relance" dans le CRM.
3. Ajouter export CSV des abonnements.
4. Ajouter recherche client par code ou telephone.
5. Ajouter stat "VIP expires aujourd'hui".
6. Ajouter statut paiement: paye, en attente, litige.
7. Ajouter un tableau "clients chauds": clic VIP sans paiement.

## Roadmap 60 jours

1. Synchroniser les tickets VIP par utilisateur Firebase.
2. Ajouter connexion admin propre.
3. Ajouter page publique de records verifies.
4. Ajouter notifications push pour alertes VIP.
5. Ajouter offres personnalisees selon segment.
6. Ajouter un module de coupons images pour WhatsApp/Telegram.
7. Ajouter tags client: gros client, reactiver, risque churn, support.

## Roadmap 90 jours

1. Integrer une API de cotes live.
2. Detecter line movements.
3. Ajouter alertes de chute de cote.
4. Ajouter Closing Line Value simplifie.
5. Ajouter comparaison de cotes par bookmaker local.
6. Ajouter leaderboard VIP optionnel.
7. Ajouter abonnement automatique si un prestataire de paiement fiable est disponible.

## Scripts de vente WhatsApp

### Relance expiration

Bonjour, ton acces VIP Goliat expire bientot. Tu peux renouveler maintenant pour ne pas rater les prochains tickets premium. Plan mensuel: 10 000 FCFA.

### Reactivation ancien VIP

Bonjour, on a relance les tickets VIP avec tracker personnel et records verifies. Offre retour disponible aujourd'hui si tu veux reprendre ton acces.

### Conversion prospect chaud

Bonjour, j'ai vu ton code VIP en attente. Tu peux finaliser ton acces et recevoir les pronos premium du jour directement apres confirmation.
